<SampleDescription>
   <p>This example shows how to use the DataGrid column resizing feature. To resize a column, click and drag the right corner of the column header.</p>
</SampleDescription>
<ActionDescription>
   <p>Columns in the Blazor DataGrid can be resized by clicking and dragging the right corner of the column header. Set the <a target='_blank' href='https://help.syncfusion.com/cr/blazor/Syncfusion.Blazor.Grids.SfGrid-1.html#Syncfusion_Blazor_Grids_SfGrid_1_AllowResizing' aria-label="Navigate to the AllowResizing property reference for DataGrid component">AllowResizing</a> property to <b>true</b> to enable column resizing. You can also prevent a specific column from being resized by setting the <a target='_blank' href='https://help.syncfusion.com/cr/blazor/Syncfusion.Blazor.Grids.GridColumn.html#Syncfusion_Blazor_Grids_GridColumn_AllowResizing' aria-label="Navigate to the AllowResizing property reference for Grid column">AllowResizing</a> property to <b>false</b>  in the column definition. Furthermore, by double clicking in the right corner of a column header, the <a target='_blank' href='https://help.syncfusion.com/cr/blazor/Syncfusion.Blazor.Grids.SfGrid-1.html#Syncfusion_Blazor_Grids_SfGrid_1_AutoFitColumnsAsync' aria-label="Navigate to the AutoFitColumnsAsync method reference for DataGrid component">AutoFitColumnsAsync</a> method will automatically adjust the column width.</p>
   <p>The resizing feature has been enabled in this demo by setting the <code>AllowResizing</code> property to <b>true</b>. The <b>Order ID</b> column can be resized between the range of <a target='_blank' href='https://help.syncfusion.com/cr/blazor/Syncfusion.Blazor.Grids.GridColumn.html#Syncfusion_Blazor_Grids_GridColumn_MinWidth' aria-label="Navigate to the MinWidth property reference for Grid column">MinWidth</a> of <b>120px</b> and <a target='_blank' href= 'https://help.syncfusion.com/cr/blazor/Syncfusion.Blazor.Grids.GridColumn.html#Syncfusion_Blazor_Grids_GridColumn_MaxWidth' aria-label="Navigate to the MaxWidth property reference for Grid column">MaxWidth</a> of <b>300px</b>. Also, column resizing has been disabled for the <b>Shipped Date</b> column.</p>
   <p>More information about the column Resizing feature can be found in this <a target='_blank' href='https://blazor.syncfusion.com/documentation/datagrid/column-resizing' aria-label="Navigate to the documentation for column resizing in DataGrid component">documentation</a> section.</p>
</ActionDescription>
